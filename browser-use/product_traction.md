# Product and Traction

Browser-use provides an open-source library and cloud infrastructure for web automation using Large Language Models (LLMs) [1](#ref-1).

## Core Product Features

*   **Web Agents**: Enables users to extract data, automate tasks, perform QA testing, and monitor websites using natural language commands. [1](#ref-1)
*   **Stealth Browsers**: Provides undetectable browser infrastructure with anti-detect features, CAPTCHA solving, and 195+ country residential proxy coverage. [1](#ref-1)
*   **Custom Models**: Purpose-built LLMs optimized for browser automation, designed for higher efficiency (tasks per dollar) compared to general models. [1](#ref-1)
*   **Skill APIs**: Converts any website into a reliable, callable API endpoint. [1](#ref-1)

## Target Users

*   **Enterprise Teams**: Used by teams at major technology and logistics companies including Anthropic, Apple, Datadog, DeepMind, DHL, FedEx, Flexport, Google, Meta, Microsoft, OpenAI, and SAP. [1](#ref-1)
*   **Developers**: Those looking for open-source libraries or cloud-based API solutions for web automation. [1](#ref-1)

## Use Cases

*   **Data Extraction**: Scraping and structured data retrieval from complex websites. [1](#ref-1)
*   **Form Automation**: Automating multi-step processes like filling out loan applications. [1](#ref-1)
*   **QA Testing & Monitoring**: Verifying website functionality and monitoring changes using natural language. [1](#ref-1)
*   **Price Monitoring**: Creating endpoints for checking prices or data across various sites. [1](#ref-1)
