# Browser-Use Overview

`browser-use` is an open-source project and platform designed to enable Large Language Models (LLMs) to interact with web browsers [1](#ref-1). It has gained significant traction on GitHub, with over 78,000 stars [1](#ref-1) and a vibrant contributor community [1](#ref-1).
