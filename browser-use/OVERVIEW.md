# Browser-Use Overview

Browser-use provides an open-source library and cloud infrastructure for web automation using Large Language Models (LLMs). [1](#ref-1)

## Key Facts
*   **Total Funding:** $17.5 million [2](#ref-2)
*   **Recent Round:** $17 million Seed Round (March 2025) [2](#ref-2)
*   **Lead Investor:** Felicis Ventures [2](#ref-2)
