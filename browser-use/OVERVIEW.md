# Browser-Use Overview

Browser-use provides an open-source library and cloud infrastructure for web automation using Large Language Models (LLMs) [1](#ref-1). It is an open-source project and platform designed to enable LLMs to interact with web browsers [2](#ref-2). It has gained significant traction on GitHub, with over 78,000 stars [2](#ref-2) and a vibrant contributor community [2](#ref-2).

## Key Facts
*   **Total Funding:** $17.5 million [3](#ref-3)
*   **Recent Round:** $17 million Seed Round (March 2025) [3](#ref-3)
*   **Lead Investor:** Felicis Ventures [3](#ref-3)
